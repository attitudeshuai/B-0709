# WordVault 划词翻译与生词本插件 (v1.0)

**WordVault** 是一款基于 Chrome Extension V3 架构的划词翻译与生词本工具。它采用 **Service Worker** 中心化数据管理，实现了划词弹窗与插件弹出的数据实时互通。

## 🏆 核心功能

1. **中文 UI 支持**: 插件弹窗及划词面板均已完全本地化为中文。
2. **长文本支持**: 选区长度限制已提升至 800 个字符，支持句子与段落翻译。
3. **友好交互反馈**: 
   - 选区获取失败、字数超出限制等错误时，增加了优雅的黑色 Toast 提示。
   - 当插件更新或重启后未识别到当前页面的环境时，会注入美观的 Toast 提醒用户“按F5刷新当前页面”。
4. **智能生词本**: 自动记录并统计单词查询次数，支持增加个人笔记。

## ⚙️ 安装与使用

1. 下载并解压代码库。
2. 在 Chrome 浏览器地址栏输入 `chrome://extensions/`。
3. 开启右上角的 **开发者模式**。
4. 点击 **加载已解压的扩展程序**，选择本项目的 `frontend` 目录。
5. **如何使用**: 在任意网页中选中一段文本（不超过800字符），按下快捷键 `Alt+Q` (Mac/Win) 即可呼出翻译面板。
   > ⚠️ **注意**: 刚安装插件后，请 **刷新** 之前已打开的网页才能生效！

## 🌐 网络与 API 说明 (重要)

本插件依赖外部免费 API 进行翻译与词典查询，由于服务服务器位于海外，因此 **网络环境** 对插件的可用性有较大影响：

1. **翻译接口**: 本插件内置 **三级智能灾备策略** 以适应复杂的网络环境：
   - **Google API** (`translate.googleapis.com`): 首选方案，速度最快。**需确保您的网络环境可以正常访问 Google 服务**，否则会触发备用方案。
   - **Google Web** (`translate.google.com`): 备用方案，抗 DNS 污染能力较强，但也**依赖国际互联网环境**。
   - **MyMemory** (`mymemory.translated.net`): 兜底方案 (完全免费，无需 Key)。
2. **词典与音标接口**:
   - 音标与例句数据来自于外部免费接口 `api.dictionaryapi.dev`。
   - ⚠️ **如果您的网络直连海外速度较慢或者被阻断，划词后可能会发现没有音标和例句**，或者整体响应时间变长，但这不影响基础的翻译功能（会降级处理）。
